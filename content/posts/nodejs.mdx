## O que é?

O nodejs é um ambiente de execução ou runtime que permite que o javascript rode no lado do servidor,o mesmo é definido por Ryan Dahl(o criador original) como um "conjunto de bibliotecas que são executadas no mecanismo V8, permitindo executar o código JavaScript no servidor".
