## O que é?

O nodejs é um ambiente de execução ou runtime que permite que o javascript rode no lado do servidor, o node utiliza por debaixo dos panos o V8 que é um compilador para códigos em javascript escrito em C++, ele implementa as especificações do ECMAscript e é multiplataforma, além disso o nodejs utiliza a libuv que é uma biblioteca escrita em C que fornece suporte para operações assincronas com base em loop de eventos, no qual o node é baseado fazendo o single threaded.
